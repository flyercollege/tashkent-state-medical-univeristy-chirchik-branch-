Statement

Accessibility Statement of Tashkent State Medical University Chirchik Branch

Tashkent State Medical University Chirchik Branch aims to provide an inclusive digital experience for users who access university information, admissions guidance, international student support, clinical practice details, academic resources and contact forms.

This website prototype is designed with accessibility principles including semantic headings, readable typography, responsive layouts, meaningful link text, alt text for important images, keyboard focus styles and reduced dependency on external resources.

Accessibility Features

  • Responsive layout for mobile, tablet and desktop users
  • Readable navigation and consistent page structure
  • Keyboard focus visibility
  • Alternative text for university logo and key images
  • Clear buttons and descriptive links
  • Form labels and placeholders for user guidance
  • High contrast navy, white, sky blue and gold color system

Ongoing Improvements

Tashkent State Medical University Chirchik Branch should continue testing the website with assistive technologies, screen readers, keyboard-only navigation and accessibility audits before official launch.

Limitations

Some demo forms, portals and media elements require production backend integration. Real documents, video captions, downloadable PDFs and interactive maps should be tested for accessibility before publication.

Feedback

If you experience difficulty using this website, please contact the university so the issue can be reviewed and improved.

Accessibility Checklist

Production accessibility tasks.

Before launch, the website should be reviewed against accessibility standards and user needs.

Check keyboard navigation on all pages
Test with screen reader software
Add captions/transcripts for videos
Ensure PDFs are accessible
Confirm form error messages are clear
Review contrast and font sizes
Add skip links where needed
Audit interactive maps and media
