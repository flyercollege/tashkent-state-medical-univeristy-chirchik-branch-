# TSMU – Chirchik Branch Premium Website

A bespoke, responsive front-end website prototype for **Tashkent State Medical University – Chirchik Branch**.

## Included

- 17 requested website pages in a fast single-page application
- Sticky navigation, top bar, mega menu, mobile menu
- Premium navy / sky blue / gold medical university design
- Animated parallax hero with video-style medical/campus scenes
- Counters, fade animations, hover cards, glassmorphism, rounded corners
- Admissions, international students, hostel, research, news, events, gallery, student life, careers, alumni, contact
- AI-powered admissions assistant demo chatbot
- Online application modal with document upload fields
- Student and faculty portal demo login
- News CMS demo using browser localStorage
- Event calendar
- Website search
- Multi-language navigation support: English, Russian, Uzbek, Arabic with RTL
- Dark mode toggle
- WhatsApp floating button and back-to-top button
- SEO meta tags and structured data
- Accessibility considerations: skip link, semantic structure, focus states, reduced-motion support

## Files

- `index.html` — app shell, SEO metadata, header, footer, modals
- `styles.css` — responsive premium design system
- `app.js` — routing, content rendering, interactions, search, CMS demo, chatbot

## Production notes

This is an offline-friendly front-end implementation. For a live production website, connect:

1. Real hero video and optimized campus media.
2. A secure backend for admissions forms and document uploads.
3. Real student/faculty SSO or LMS authentication.
4. A production CMS for news, events, gallery, translations, and approvals.
5. Google Maps iframe/API with the official campus location.
6. Official university contact details, fee table, accreditations, and leadership bios.
